# Abraham Pérez — Portfolio

Personal portfolio website built with Next.js 14, Tailwind CSS, and next-intl.

**Live:** [abrahamperez.net](https://abrahamperez.net)

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- next-intl (ES, EN, FR, DE)

## Run locally

```bash
npm install
npm run dev
```
